# Asteroids!

I created this version of the vintage arcade game 'Asteroids!' in C++ with the SFML library for fun!

# Commands

- **Boost**: up arrow
- **Slow**: down arrow
- **Rotate Left**: left arrow
- **Rotate Right**: right arrow
- **Fire Laser**: spacebar
- **Close Window**: 'q'
- **Start/Restart Game**: 's'
